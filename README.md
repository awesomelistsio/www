# Awesome Lists Jekyll Site

This package converts the Awesome Lists static site into a Jekyll-powered website with blog support.

## Features

- Jekyll structure
- Blog support with `_posts`
- Existing MDL-inspired design retained
- Existing assets and JavaScript retained
- Real Awesome Lists dataset retained
- GitHub Pages compatible

## Run locally

```bash
bundle install
bundle exec jekyll serve
```

Then open:

```text
http://127.0.0.1:4000
```
